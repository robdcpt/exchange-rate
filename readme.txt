

Exchange Rates Sample PHP Web App

This is a simple demo app integrating a PHP Web App to a MySQL Database and with a 3rd Party Exchange Rates API https://exchangeratesapi.io/

Pre-requisites
===============
Docker / Docker-compose

To run the web application;

1. clone the project
2. cd into the root directory
3. type docker-compose up
4. point your web browser to http://localhost/
